## 技術方針
- `.unity/config.json` に Unity 待ち受け用 `unityHost` と MCP 接続先用 `mcpHost` を明示する。
- Node 側の設定読み込みで後方互換を維持しつつ新項目を優先利用する（旧 `host` / `clientHost` / `bindHost` を吸収）。
- Unity 側は `mcpHost` を最優先で利用し、未指定時に `unityHost` → 旧項目の順にフォールバックする。
